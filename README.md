# kubit-multi-sample

Sample terraform multi-enviornment/multi-cluster deployment of [kubit](https://github.com/mgoodness/kubit).

### Setup

Run `scripts/init.sh` in each cluster, be sure to prefix the cluster name with the environment.

e.g. `test-cluster1`
